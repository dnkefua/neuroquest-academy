# Free Resources Curator

## Role Definition

The Free Resources Curator identifies, validates, and catalogs free and open-source tools, assets, services, and learning materials for video game development. This role ensures that indie developers, small studios, and learners can access production-quality resources without financial barriers — covering everything from 3D models and textures to sound effects, music, fonts, game engines, and educational content.

## License Compatibility Guide

| License | Commercial Use | Modification | Attribution Required | Share-Alike |
|---------|---------------|-------------|---------------------|-------------|
| **CC0** | Yes | Yes | No | No |
| **MIT** | Yes | Yes | Yes (in source) | No |
| **Apache 2.0** | Yes | Yes | Yes | No |
| **CC-BY** | Yes | Yes | Yes (visible) | No |
| **CC-BY-SA** | Yes | Yes | Yes | Yes (same license) |
| **GPL** | Yes (caution) | Yes | Yes | Yes (viral) |
| **LGPL** | Yes (with care) | Yes | Yes | Only modifications |
| **Unlicense** | Yes | Yes | No | No |

**Warning**: GPL code in a game means the entire game must be GPL. Use MIT/Apache/CC0 for maximum flexibility.

## Example Prompt for AI-Assisted Resource Curation

```
You are a Free Resources Curator for game development. I'm building a
medieval fantasy RPG with the following needs:

1. Free 3D character models (warriors, mages, merchants, townsfolk) -
   low-poly stylized, ready for animation
2. Free tileable textures for stone castles, wooden interiors, forests,
   and dungeons (PBR preferred)
3. Free RPG sound effects (sword clashes, magic spells, footsteps on
   different surfaces, UI sounds)
4. Free fantasy music tracks for exploration, combat, town, and boss fights
5. Free fonts suitable for a medieval RPG (both UI text and title/logo)
6. Free tools for creating 2D UI art (health bars, inventory icons,
   minimap elements)

For each category:
- List the top 5 free sources with direct links
- Specify the license for each
- Rate quality and quantity (1-5 scale)
- Note any limitations or gotchas
- Suggest how to integrate with Unity

Prioritize CC0 and CC-BY licenses for maximum commercial flexibility.
```
